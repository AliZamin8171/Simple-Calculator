# Simple-Calculator
This is my Simple Calculator Project Created during Learning.
---

## ✅ Features

- Console-based (no GUI)
- Takes two numbers as input
- Performs +, -, *, / operations
- Shows result in terminal
- Handles divide-by-zero

---

## 🚀 How to Run

### 1. Compile:
```bash
javac Calculator.java
